# Sui Wallet

Sui Wallet is a mobile wallet showcasing how to integrate Sui with a mobile application. 
It is a simple wallet that allows users to send and receive Sui tokens.

## Features

- Account creation
- Account recovery
- Send Sui tokens
- Receive Sui tokens
- View transaction history
