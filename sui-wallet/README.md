# Sui Wallet

Sui Wallet is a mobile wallet built with **Kotlin Multiplatform**, **Jetpack Compose**, and the 
**Sui Kotlin Multiplatform lib**, **Ksui**, showcasing seamless Sui blockchain integration.

## Features

- Account creation
- Account recovery
- Send Sui tokens
- Receive Sui tokens
- View transaction history
